## Different Kinds of Types

There are various kinds of built-in types in ADAN, which consist of:

### Character-based Types

- String
- Char

### Number-based Types

- Number (Handles *both* Integers and Floating-point values automatically)
- Integer
- Float

### Array Types

To create an array in ADAN you can use the basic syntax: `<Type>[]`. This creates a simple 1-dimensional array.

This process can be stacked to create higher dimensions of arrays, such as `<Type>[][]`, or `<Type>[][][]`, and so on.

### Other Types

- Bool
- None
- Void
- Auto
